Adds a field to configure spacing - margin and padding.

== Description ==

An ACF PRO field to configure styles for a section including:

* Padding
* Margin

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `acf-spacing` folder into your `wp-content/plugins` folder
2. Activate the acf-spacing plugin via the plugins admin page
3. Create a new field via ACF and select the Section Styles type
4. Please refer to the description for more info regarding the field type settings
